# Telegram-бот со статистикой

Простой Telegram-бот на aiogram: здоровается, считает визиты и отдаёт статистику по пользователям.

## Что умеет

- `/start` — приветствует пользователя и показывает, сколько раз он уже запускал бота (количество визитов).
- `/stats` — показывает счётчик визитов конкретного пользователя и общее количество уникальных пользователей бота.
- `/top` — выводит топ-5 пользователей по количеству вызовов `/start` (если такие пользователи есть).
- Любой обычный текст — отвечает тем же сообщением (режим «эхо»).

Под капотом бот ведёт учёт уникальных пользователей и количества их визитов в базе данных SQLite.

## Стек

- Python
- [aiogram](https://docs.aiogram.dev/) 3.28.2
- SQLite — модуль `sqlite3` из стандартной библиотеки Python (отдельная установка не требуется)

## Как запустить

1. Клонируй репозиторий и перейди в папку проекта:

   ```bash
   git clone <ссылка-на-репозиторий>
   cd week_05_bot
   ```

2. Создай и активируй виртуальное окружение:

   ```bash
   python -m venv .venv
   ```

   - Linux / macOS:

     ```bash
     source .venv/bin/activate
     ```

   - Windows:

     ```bash
     .venv\Scripts\activate
     ```

3. Установи зависимости:

   ```bash
   pip install -r requirements.txt
   ```

4. **Настрой токен.** В репозитории нет файла `config.py` (он в `.gitignore`, чтобы токен не попал в открытый доступ). Вместо него лежит шаблон `config.example.py`. Скопируй его в `config.py` и впиши свой токен:

   - Linux / macOS:

     ```bash
     cp config.example.py config.py
     ```

   - Windows:

     ```bash
     copy config.example.py config.py
     ```

   Затем открой `config.py` и вставь токен, который выдаёт [@BotFather](https://t.me/BotFather).

   > Без этого шага бот не запустится — `config.py` обязателен.

5. Запусти бота:

   ```bash
   python main.py
   ```

   > База данных SQLite (`visits.db`) создаётся автоматически при первом запуске — никаких дополнительных действий не требуется. Файл базы добавлен в `.gitignore`, поэтому в репозиторий не попадает.
